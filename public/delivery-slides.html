システム概要

受講生・コーチ・管理者の3つのロールに対応した英語学習プラットフォーム。単語帳、文法、発音練習、課題管理、スケジュール管理機能を統合し、効率的な学習体験を提供します。

FEATURE 01

認証・ユーザー管理システム

仕様

  • NextAuth.jsによるセキュア認証
  • 3つのロール（受講生・コーチ・管理者）
  • ロールベースのアクセス制御
  • セッション管理・自動リダイレクト
  • bcryptによるパスワードハッシュ化

実装詳細

認証方式Credentials Provider
セッションJWT Token
パスワードbcrypt (salt rounds: 10)
保護ルートMiddleware + Server Auth
FEATURE 02

ダッシュボード

仕様

  • 週間・月間学習時間の可視化
  • 学習内訳（単語・文法・発音・課題）
  • 目標進捗の表示
  • 予定されたセッション一覧
  • 未完了課題の確認

データモデル

StudyLog学習記録（日時・時間・種別）
Goal目標（週間・月間・最終）
CalendarEventスケジュールイベント
AssignmentState課題の進捗状態
FEATURE 03

単語帳機能

仕様

  • デッキ管理（複数単語帳対応）
  • フラッシュカード形式の学習モード
  • テストモード（意味入力式）
  • 進捗トラッキング（新規・学習中・習得）
  • レベル別分類（初級・中級・上級）

データ構造

VocabDeckデッキ情報・カード配列
VocabCard単語・意味・発音・例文
VocabProgressユーザー別学習進捗

正解/不正解回数に基づく自動ステータス更新

FEATURE 04

文法学習機能

仕様

  • レッスンコンテンツ表示（Markdown対応）
  • 4択クイズ形式のテスト
  • 解説表示機能
  • スコア記録・進捗管理
  • レベル別カリキュラム

データ構造

GrammarLessonタイトル・内容・問題配列
GrammarQuestion問題文・選択肢・正解・解説
GrammarProgress完了状態・スコア
FEATURE 05

発音練習機能

仕様

  • ターゲットテキスト表示
  • カテゴリ別分類（日常会話・ビジネス等）
  • レベル別難易度設定
  • 練習履歴の記録

データ構造

PronunciationExercise練習課題情報
targetText練習対象テキスト
categoryカテゴリ分類
level難易度レベル
FEATURE 06

カレンダー・スケジュール管理

仕様

  • 月間カレンダービュー
  • イベント種別（コーチング・学習・課題）
  • 日程変更リクエスト機能
  • コーチとのセッション管理
  • ステータス管理（予定・完了・キャンセル）

データ構造

CalendarEventイベント情報
typecoaching / study / assignment
changeRequest変更リクエスト情報
statusscheduled / completed / cancelled

  • スコア記録
