生徒機能

📚

単語学習

フラッシュカード形式の単語帳機能。進捗状況（new/learning/mastered）を自動追跡し、復習タイミングを最適化。テスト機能とヒントシステム搭載。

📝

文法学習

体系的な文法レッスンと動画解説。各レッスン終了後のクイズで理解度を確認。正答率と完了状況を記録。

🎤

発音練習

音声録音と評価機能。正確性・流暢性・発音の3軸でスコアリング。音素レベルでの詳細フィードバック提供。

📊

ダッシュボード

週間/月間の学習時間可視化。コンテンツ別学習内訳、未完了課題数、次回コーチングセッションを一目で確認。

🎯

目標設定

週間・月間・最終目標の3タイプ。目標時間と達成状況を追跡。モチベーション維持をサポート。

📅

カレンダー

コーチングセッションの予定確認。日程変更リクエスト機能搭載。承認ワークフローで柔軟に対応。

👨‍🏫

コーチ機能

  • ✓

    担当生徒管理

    担当する生徒の一覧表示。各生徒の週間学習時間、課題提出状況をリアルタイムで確認。

  • ✓

    生徒詳細ページ

    個別生徒の学習進捗を詳細表示。単語・文法・発音の各分野での達成状況を可視化。

  • ✓

    コーチングセッション管理

    セッションのスケジュール作成・編集。カレンダービューで予定を一元管理。

  • ✓

    日程変更リクエスト対応

    生徒からの日程変更リクエストを承認/却下。ペンディング件数をダッシュボードで通知。

  • ✓

    コーチダッシュボード

    担当生徒数、今週のセッション予定、変更リクエスト件数を集約表示。

⚙️

管理者機能

  • ✓

    ユーザー管理

    全ユーザーの作成・編集・一覧表示。ロール設定（生徒/コーチ/管理者）とコーチ割り当て。

  • ✓

    生徒・コーチマッチング

    未割り当ての生徒とコーチを表示。ワンクリックでマッチング、双方に通知自動送信。

  • ✓

    教材管理

    単語デッキ、文法レッスン、発音エクササイズの作成・編集。レベル設定と公開管理。

  • ✓

    課題管理

    課題の作成・割り当て・進捗確認。期限設定とステータス管理機能。

  • ✓

    データエクスポート

    ユーザーデータ、学習ログ、課題提出データをJSON/CSV形式で出力。日付範囲フィルター対応。

  • ✓

    管理者ダッシュボード

    全体統計（生徒数、コーチ数、学習時間）、今月の新規登録、課題完了率を集約表示。

🔧

共通機能・システム

🔐

認証・認可

NextAuth v5によるJWTベース認証。bcryptによるパスワードハッシュ化。ロールベースのアクセス制御。

🔔

通知システム

5種類の通知タイプ（課題、コーチング、フィードバック、システム、リマインダー）。既読管理機能搭載。

📈

学習ログ

学習活動の自動記録。コンテンツ別（単語/文法/発音/課題）の学習時間追跡。

📱

レスポンシブ対応

PC・タブレット・スマートフォンに対応したレスポンシブデザイン。モバイルメニュー搭載。
